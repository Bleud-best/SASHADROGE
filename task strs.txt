Задание на строки: 

1. Написать функцию с двумя параметрами: 
	1й параметр(строка, любой произвольный текст), в нем вы будете производить поиск
	2й параметр(любой литерал из number/string), в нем указано что вы будете искать
Функция проверяет строку и если в ней встречается то, что передано как второй параметр - 
выводите в консоль сообщение с ошибкой, указав позицию в строке, где было найдено совпадение. 
Если совпадений нет - выдаст в конце проверки сообщение об успехе. 

Пример:
  myFunc('text 1apple', 1) // err, потому что в строке есть символ 1 
  myFunc('text 1apple', 'e') // err, потому что в строке есть символ 'e'
  myFunc('text 1apple', '+') // ok, потому что в строке нету символа '+'


2. Скопировать предыдущее решение и улучшить его: 
теперь во второй параметр можно паредать не 1 символ, 
а несколько(все они идут слитно). Функция будет проверять
 на наличие любого символа из переданной группы.

Пример:
  myFunc('text 1apple', 'a1&') // err 
  myFunc('text 1apple', 't3!') // err
  myFunc('text 1apple', 'wk6') // ok


3. Скопировать предыдущее решение и улучшить его: сделайте так, 
чтобы во втором параметре можно было передать параметры через пробел и все работало так же.

Пример:
  myFunc('text 1apple', 'a 1 &') // err 
  myFunc('text 1apple', 't 3 !') // err
  myFunc('text 1apple', 'w k 6') // ok


4. Создать функцию, которая принимает 2 параметра. 1й - строка(где искать), 
2й - строка(что искать). Функция должна вернуть кол-во найденных совпадений.

Пример:
  myFunc('text 1apple', 'p') // 2
  myFunc('text 1apple', 'pp') // 1
  myFunc('text 1apple', 'r') // 0


5. Создать функцию, которая принимает 2 параметра. 1й - строка(где искать), 
2й - строка(что искать). Функция должна вернуть строку со всеми индексами найденных 
совпадений через пробел или пустую строку, если совпадений нет.

Пример:
  myFunc('text 1apple', 'p') // "7 8"
  myFunc('text 1apple', 'pp') // "1 10"
  myFunc('text 1apple', 'r') // ""


6. Создайте функцию, которая принимает 2 параметра. 1й - строка, 2й - число. 
Ваша функция должна вернуть полученную строку, в которой переведет все символы 
в верхний регистр шагом указанным во втором параметре

Пример:
  myFunc('text 1apple', 2) // "teXT 1APplE"
  myFunc('text 1apple', 5) // "teXT 1APPLe"
  myFunc('text 1apple', 0) // "TEXT 1APPLE"


7. Реализуйте свою функцию "slice", работает так же как и метод строк, 
принимает 3 параметра - строку, индекс начала и индекс конца 

Пример:
  myFunc('text 1apple', 2, 4) // "xt"
  myFunc('text 1apple', 0) // "text 1apple"
  myFunc('text 1apple', -2) // "le"
